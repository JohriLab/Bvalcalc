The following files in this shared directory have code to reproduce the B-value calculator (Bvalcalc) manuscript results:

The code used to generate Figure 2 is available at: https://github.com/JohriLab/Bvalcalc/blob/main/manuscript/Figure2.rmd.

The code used to generate Figure 3 is available at: https://github.com/JohriLab/Bvalcalc/blob/main/manuscript/Figure3.py.

See https://github.com/JohriLab/Bvalcalc/blob/main/manuscript/Bvalcalc_code.sh for a full list of Bvalcalc commands used to generate results, including Figure 4 and the B-maps.

B-value maps across autosomes that we have calculated using template parameters are available for download at Zenodo: https://doi.org/10.5281/zenodo.17334918.
